# Workflows Métier par Type d'Activité — HUB_RESA

## 1. Transport (Bus)
**Modules dashboard :**
- Tableau de bord : départs du jour, billets vendus, recettes
- Gestion des départs (horaires, bus, lignes)
- Billetterie / Caisse (vente de billets, encaissement)
- Gestion des bus (flotte)
- Gestion des lignes (trajets)
- Gestion des gares (stations)
- Clients / Réservations
- Finance (recettes, crédits, facturation)
- Messages HUB_RESA
- Configuration

## 2. Agence de Voyage (Avion)
**Modules dashboard :**
- Tableau de bord : vols du jour, billets vendus, recettes
- Recherche de vols (API compagnies aériennes)
- Billetterie avion (émission de billets, e-tickets)
- Caisse (encaissement billets, frais de service)
- Gestion des destinations
- Clients / Réservations
- Finance (commissions, recettes)
- Messages HUB_RESA
- Configuration

## 3. Restauration
**Modules dashboard :**
- Tableau de bord : commandes du jour, chiffre d'affaires, tables occupées
- Gestion du menu (plats, boissons, prix)
- Commandes (nouvelles, en cours, servies)
- Tickets de caisse (POS - Point of Sale)
- Gestion des tables
- Clients / Fidélité
- Finance (recettes, dépenses, bénéfices)
- Messages HUB_RESA
- Configuration

## 4. Expédition (Shipping/Colis)
**Modules dashboard :**
- Tableau de bord : colis du jour, en transit, livrés
- Nouveau colis (enregistrement, pesée, tarification)
- Suivi des colis (statuts : reçu, en transit, livré)
- Bons de livraison / Reçus
- Caisse (encaissement frais d'expédition)
- Clients / Destinataires
- Finance (recettes, tarifs)
- Messages HUB_RESA
- Configuration

## 5. Hôtel
**Modules dashboard :**
- Tableau de bord : chambres occupées/libres, arrivées/départs du jour
- Gestion des chambres (types, tarifs, statuts)
- Réservations (check-in, check-out)
- Services additionnels : Restauration, Pressing, Piscine, Spa
- Caisse (facturation client, encaissement)
- Clients / Historique séjours
- Finance (recettes par service)
- Messages HUB_RESA
- Configuration

## 6. Boutique
**Modules dashboard :**
- Tableau de bord : ventes du jour, stock faible, chiffre d'affaires
- Catalogue produits (articles, prix, stock)
- Ventes / Caisse (POS)
- Gestion du stock (entrées, sorties, inventaire)
- Clients / Fidélité
- Finance (recettes, marges)
- Messages HUB_RESA
- Configuration
